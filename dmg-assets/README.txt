BigEcho

Первый запуск
-------------

Приложение пока не подписано сертификатом Apple Developer ID и не прошло нотаризацию, поэтому macOS помечает его «карантинным» и Gatekeeper может показать сообщение о том, что приложение «повреждено» или «не может быть открыто».

Инструкция
----------

1. Перетащите BigEcho.app в папку Applications.
2. Перед первым открытием выполните в терминале команду:

       xattr -cr /Applications/BigEcho.app

3. Запустите приложение обычным способом.

Команда снимает с приложения атрибут карантина (com.apple.quarantine). Её достаточно выполнить один раз — после обновлений повторять не нужно, если вы заменяете .app из нового DMG, то команду нужно будет выполнить снова.
